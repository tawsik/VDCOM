Задание на моделирование БД
 

Спроектируй базу данных для учёта продаж мороженного клиентам.

Минимальный набор сущностей, который должен храниться в нашей БД:

Клиент
Продажи
Товар
В качестве результата предоставь схему БД. На схеме должны быть представлены:

Названия таблиц и необходимых полей (названия давать осмысленные, однородные).
Все необходимые ограничения: типы данных, ключи и т.п.
Связи между таблицами.
Связи должны быть подписаны.
Схему можно отрисовать в draw.io.
Дополнительным плюсом будет скрипт создания базы данных (под Microsoft SQL Server).
